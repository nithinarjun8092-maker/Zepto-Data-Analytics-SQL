# Zepto Data Analytics | SQL Project

## Overview
End-to-end SQL data analytics project on Zepto product 
data covering pricing, discounts, inventory and revenue 
analysis using MySQL.

## Database & Table
- Database: zepto_project
- Table: zepto (10 columns including category, name, 
  mrp, discount, quantity, stock status)

## What I Did
- Created database and table from scratch
- Performed data exploration and null value checks
- Cleaned data — removed zero price products
- Converted prices from paise to rupees

## Business Questions Answered
1. Top 10 best value products by discount percentage
2. High MRP products that are out of stock
3. Estimated revenue calculation per category
4. Products with high MRP but low discount
5. Top 5 categories with highest average discount
6. Price per gram analysis for best value products
7. Product weight categorisation (Low, Medium, Bulk)
8. Total inventory weight per category

## Tools Used
- MySQL
- SQL (DDL, DML, Aggregations, Case Statements, 
  Group By, Order By)

## Key Insights
- Identified top discount categories for customers
- Found high value out of stock products
- Calculated estimated revenue per category
- Analysed price per gram for best value products
